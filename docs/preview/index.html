What happens when you click run

  1. Generate search queries from your context The page first asks Gemini to turn your project description into topic keywords plus arXiv and PubMed queries.
  2. Retrieve recent candidates It pulls live recent papers from arXiv and PubMed inside a fixed first-run window.
  3. Rank the candidates with Gemini The page then asks Gemini to score relevance and write the short digest explanations.
  4. Open the email-style HTML in a new tab You see the digest output immediately, but nothing is sent by email from this page.

Fixed for a fast first impression: 5 selected papers, last 168 hours, real retrieval from arXiv and PubMed, no email send.
