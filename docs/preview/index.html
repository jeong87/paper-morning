What happens when you click run

  1. Generate search queries from your context The page first asks Gemini to turn your project description into topic keywords plus arXiv and PubMed queries.
  2. Retrieve candidates for your chosen search intent What's New prefers recent work, while Best Match and Discovery search wider horizons for stronger context fit.
  3. Rank the candidates with Gemini The page then asks Gemini to score relevance and write short explanations for why each paper matters.
  4. Open the result page in a new tab You see the ranked output immediately, but nothing is sent by email from this page.
Live search scope

Fast but adaptive: 5 selected papers, real retrieval from arXiv and PubMed, explicit intent and horizon controls, no email send.

Browser limitations mean this page focuses on the fastest no-install path. The local install can add more operational controls later, but this live search should already make the product value clear.
